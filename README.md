# Chipped-Driven
In a prehistoric world ruled by monsters, muscle isn't enough. Play as a caveman inventor crafting the world's first steam-powered stone machines. Build a Mobile Steam Haven—a walking fortress of bone and vine—to cross deadly biomes. Scavenge materials, fight apex predators with pneumatic tools, and find a safe land to build a new civilization.
